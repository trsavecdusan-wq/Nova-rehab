NAVODILA ZA GITHUB BUILD

1. Vsebino mape Nova-rehab-fixed nalozi v root GitHub repozitorija.
2. Pojdi na GitHub > Actions > Android APK Build.
3. Klikni Run workflow ali naredi push na main/master.
4. Po uspesnem buildu prenesi artifact: NovaRehab-debug-apk.

POPRAVEK V TEJ VERZIJI:
- prejsnji build je padel, ker se Gradle ZIP razpakira v mapo gradle-8.1.1, skripta pa je iskala gradle-8.1.1-bin.
- gradlew je popravljen, da pravilno klice ~/.gradle/rehab-wrapper/gradle-8.1.1/bin/gradle.

Ce build pade, poslji screenshot napake iz sekcije Build debug APK.
